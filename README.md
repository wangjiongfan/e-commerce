### 使用说明
如果有本地服务器可以运行 数据库 与 php 文件的可以使用完整版，完整版通过连接后端与数据库，实现模拟全栈开发的效果。
完整版使用说明详情请看完整版文件夹内的使用说明。



如果没有本地服务器可以使用单机版，单机版只需要在装有nodejs的环境变量下用 终端 或 cmd窗口 在项目文件夹下使用 npm install --save 给项目安装脚手架并使用 npm run serve 获取热加载地址就可以快速运行。



单机版与完整版的jingdong文件夹下都有一个已经完成打包的dist文件夹，将文件夹放入本地服务器就可以用浏览器快速浏览。
